# Stillforge

Handcrafted melodic steel drums forged from propane tanks. Interactive playable drum, commission flow, craft gallery, and a deep SEO guide library.

## Stack

- React 19 + Vite 8
- React Router 7
- Web Audio API playable drum
- Vercel serverless `/api/health`, `/api/inquire`, `/api/subscribe`

## Local

```bash
npm install
npm run dev
```

## Deploy

Vercel project with `framework: vite`, `buildCommand: vite build`, `outputDirectory: dist`.

Optional env: `SUPABASE_URL`, `SUPABASE_SERVICE_ROLE_KEY` for persisting inquiries/subscribers.

## Brand

Stillforge - forged from fire, tuned for stillness.
